Job Description


  • We are looking for a React Native developer interested in building performant mobile apps on both the iOS and Android platforms. You will be responsible for architecting and building these applications, as well as coordinating with the teams responsible for other layers of the product infrastructure. Building a product is a highly collaborative effort, and as such, a strong team player with a commitment to perfection is required.

Responsibilities

  • Build pixel-perfect, buttery smooth UIs across both mobile platforms.
  • Leverage native APIs for deep integrations with both platforms.
  • Diagnose and fix bugs and performance bottlenecks for performance that feels native.
  • Reach out to the open source community to encourage and help implement mission-critical
  • software fixes—React Native moves fast and often breaks things.
  • Maintain code and write automated tests to ensure the product is of the highest quality.
  • Must have basic understanding of developer account management, provisioning profile and certificates, and publishing mobile applications in different APP Play Stores.

Skills

  • Firm grasp of the JavaScript (and TypeScript or ClojureScript) language and its nuances, including ES6+ syntax
  • Knowledge of Object-oriented programming and follows mobile design patterns and guidelines
  • Ability to write well-documented, clean Javascript code
  • Rock solid at working with third-party dependencies and debugging dependency conflicts
  • Familiarity with native build tools, like XCode, Gradle (Android Studio, IntelliJ)
  • Understanding of REST APIs, the document request model, and offline storage
  • Experience with automated testing suites and any CI/CD tools.
  • Experience in interfacing mobile applications with hardware devices (Wifi / Bluetooth Smart), embedded systems. Knowledge of low-level C-based libraries is a plus.
  • Experience with other cross platform framework, like Google Flutter, is a plus

Requirements

  • Candidate must possess at least Diploma / Bachelor's Degree in Computer Science or its equivalent.
  • At least 2 - 4 Year(s) of working experience in the related field is required for this position.
  • Required Skill(s): android studio, XCode, gradle, automated testing suites, REST API, mobile embedded systems, mobile design patterns, react native, JavaScript, typescript, clojurescript, es6+, syntax
